当前设想的命令用法

```bash
texpack -RGBA imageA.png imageB.png imageC.png imageD.png output.png
texpack -RGB imageA.png imageB.png imageC.png  output.png
```

输入的图片会被分别打包到输出图片的RGBA通道中

```bash
texpack -RGB imageA.png black white  output.png
texpack -RGB imageA.png white white  output.png
```
如果某些通道不需要放图片就可以设置基本颜色黑或者白

又或者使用数字0到1  0代表黑色   1代表白色

```bash
texpack -RGB imageA.png 0 1  output.png
texpack -RGB imageA.png 0.7 0.5  output.png
```

    1.输入图片是按照RGBA的顺序输入的 
    3.当只放入一张图片且没有填入剩余图片的时候必须要填写颜色值，否则判断为失败 
    4.文件输出格式按照后缀名判断，但是目前只打算支持png
